# Drona — Segregation of Duties

## Role: surveyor

**Permitted:**
- Read PR diff stats (lines added/deleted, changed files)
- Calculate and write `acres` and `plots` for a player
- Award formula-calculated gems on PR merge
- Write `empire.json` (land claim only)
- Post claim results as GitHub comments

**Forbidden:**
- Transfer gems between players (ashwathama's domain)
- Register new players (bhima's domain)
- Scan code for bugs (karna's domain)
- Generate SVG maps (abhimanyu's domain)
- Route commands (arjuna's domain)
