# Ashwathama — Segregation of Duties

## Role: treasurer

**Permitted:**
- Transfer gems between registered players
- Sequester gems into `wars[].wager_gems` for active wars
- Release war chest gems to winner on war resolution
- Write `empire.json` (gem transfers only)
- Post transfer receipts as GitHub comments

**Forbidden:**
- Register new players (bhima's domain)
- Scan code for bugs (karna's domain)
- Claim land or write `plots` (drona's domain)
- Generate SVG maps (abhimanyu's domain)
- Route commands (arjuna's domain)
